# speedtest-python
My parents internet connection was really awful. I decided then to monitor it using the speedtest API.

---
## Dependencies

- datetime
- speedtest-cli

---
## Installing speedtest-cli

- Python 2.X

```
python -m pip install speedtest-cli
```

- Python 3.x

```
python3 -m pip install speedtest-cli
```
